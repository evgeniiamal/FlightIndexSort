# FlightIndexSort

## Описание
Консольное приложение на языке C++, предназначенное для обработки информации о рейсах воздушных судов.
Программа считывает данные из текстового файла, выполняет проверку корректности каждой записи, сообщает об обнаруженных ошибках, формирует индексный массив и сортирует записи по времени входа воздушного судна в зону без изменения исходного массива данных.

Проект разработан в рамках изучения алгоритмов сортировки, структур данных и файлового ввода-вывода.

## Основные возможности

- чтение данных о рейсах из текстового файла
- проверка существования входного файла
- валидация формата каждой записи
- контроль корректности
- обработка ошибок с указанием номера строки
- индексная сортировка по времени
- вывод отсортированных данных в табличном виде

## Используемые технологии

- C++
- Структуры данных
- Работа с файлами
- Потоковый ввод
- Индексная сортировка
- Форматированный вывод

## Структура входных данных

Каждая запись должна содержать четыре поля:

```
Время  Модель_ЛА  Бортовой_номер  Номер_рейса
```

Пример:

```text
08:20 Boeing737 R-1234 1201
11:45 AirbusA320 Б-5678 4310
09:15 SuperJet100 A-9012 2154
```

## Алгоритм работы

1. Чтение входного файла.
2. Проверка корректности каждой записи.
3. Формирование массива структур `Flight`.
4. Создание индексного массива.
5. Сортировка индексного массива методом выбора.
6. Вывод записей в отсортированном порядке.

Использование индексной сортировки позволяет сохранить исходный массив без изменений.
